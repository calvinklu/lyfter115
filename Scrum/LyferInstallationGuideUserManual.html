Installation Guide/User Manual for Lyfter

Overview: Lyfter is an Android App for Android users, which helps to plan and notify you of your workout schedule.

Installation: Download and run Android Studio - https://developer.android.com/studio/

  1. Manual Download
  • Go to app location on github - https://github.com/calvinklu/lyfter115 
  • Download ZIP of application
  • Unzip folder
  • Open Android Studio and start a new project
  • Go to File → New → Import Project
  • Find the project and click OK.
  • Allow the gradle to build (this may take a few minutes)
  • Run the app (Shift + F10)
  • If no virtual devices is setup, click on Create New Virtual Device
  • Select Pixel 2 XL and click Next
  • Select any system image (Pie, Oreo, etc), this will take time to download as well
  • Click Next → click Finish
  • A virtual device should appear under Connected Devices when you try to run the app again
  • Select virtual device (Pixel 2 XL API 28) and click OK
  • The app will build again (will take a few minutes) and an emulator will popup
  • Once the app is done building, run the app
  • Select virtual device
  • The Lyfter app should automatically load up on the emulator
  • Lyfter is now ready for use.
